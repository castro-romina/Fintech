# 💳 FinTech Wallet

Billetera virtual desarrollada como proyecto educativo para **Educación IT**.  
Permite registrar usuarios, gestionar cuentas y realizar depósitos y retiros desde el navegador, sin necesidad de backend.

---

## 🚀 Tecnologías

- HTML5
- CSS3
- JavaScript (Vanilla)
- Vite

---

## ✨ Funcionalidades

- **Registro e inicio de sesión** de usuarios
- **Depósitos y retiros** con validación de saldo
- **Historial de movimientos** por cuenta
- **Panel de administración** para gestionar clientes, cuentas y movimientos

## 📦 Instalación y uso

```bash
# Clonar el repositorio
git clone https://github.com/castro-romina/Fintech.git 

# Entrar al proyecto
cd fintech

# Instalar dependencias
npm install

# Iniciar en desarrollo
npm run dev
```
